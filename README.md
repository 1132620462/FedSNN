# Federated Learning with Spiking Neural Networks

## Requirements
python>=3.7
pytorch>=1.7.1

## Run

For example:
> python main_fed.py --dataset mnist --iid --num_channels 1 --model cnn --epochs 50 --gpu 0  

NB: for CIFAR-10, `num_channels` must be 3.
